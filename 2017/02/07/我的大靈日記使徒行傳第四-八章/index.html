我的大靈日記使徒行傳第四~八章

高宇哲 2月 07, 2017

使徒行傳4:32~5:16細研

觀察

  1. 4:32~4:35 初代教會凡物公用的光景
  2. 4:36~4:37 巴拿巴是好榜樣
  3. 5:1~5:16 亞夫婦是壞榜樣

解釋

Question

  1. 難道上帝支持共產主義嗎?
  2. "其餘的人"指的是誰?(5:13)
  3. 為甚麼要凡物公用?

Ans

  1. 當時的時代背景是猶太教朝聖者回來過節時信耶穌的， 很多人東西沒有帶很夠，因此需要凡物公用。
  2. 5:12的"他們"可以翻成"信的人"，因此"其餘的人"就是"不信的人"。
  3. 同1.

歸納

  1. 不要欺騙聖靈
  2. 上帝很有原則
  3. 重視教會的聖潔
  4. 彼此相愛，看見別人需要
  5. 甘心樂意獻上所有
  6. 誠實面對上帝

應用

看見別人有需要時就要捨己

使徒行傳5:17~5:31細研

提要

大祭司和他的一切同人，就是撒都該教門的人抓住使徒 主的使者晚上放他們出來，再進殿裡教訓人。守殿官和差役 再次把使徒帶回來質問，使徒們回答順從神，不順從人，是應當的。 公會的人惱羞想殺了使徒，而迦瑪列持平的評論。公會釋放使徒。

-->迦瑪列: 保羅的老師

使徒行傳6:1~6:15細研

解釋

Question

  1. 司提反不是管飯食的嗎?(v2)怎麼後來去"在民間行了大奇事和神蹟"(v8)?
  2. 為甚麼百姓、長老，並文士說"我們曾聽見他說，這拿撒勒人耶穌要毀壞此地，也要改變摩西所交給我們的規條"?(v14)

Ans

  1. 能夠傳福音的不只使徒
  2. 是毀謗，公會的人以路21:6耶穌說過的話斷章取義，另外太5:18說道耶穌來是要成全律法。

歸納

  1. 能夠出去傳福音的不只是牧師
  2. 學會管理信仰群體是一件重要的事情

應用

禱告求神賜下管理的智慧

使徒行傳第八章

觀察

  1. v1~v8 司提反事件後的交代
  2. v9~v11 西門行邪術
  3. v12~v13 西門信主
  4. v14~v17 彼得約翰為撒瑪利亞人按手
  5. v18~v24 彼得教訓西門
  6. v25 使徒回程並傳福音
  7. v26~v40 腓力傳福音給太監

解釋

Question

  1. v15~v16為甚麼要按手?
  2. 為甚麼路加突然在這裡記腓力傳福音給太監?
  3. 為甚麼路加不直接從司提反事件(7:60)接到掃羅信主，而要插入這段?

Ans

  1. 一種象徵 --> 參考司托得<當代聖靈工作>
  2. 太監是一個有大權的人(v27)，代表福音要廣傳了而非限縮在猶太全地。 另外路加的寫作對象是提阿非羅大人，路加想引領提阿非羅知道福音是這樣傳揚到外邦人的。
  3. 目前福音已經傳到撒瑪利亞，呼應1:8。

歸納

  1. 不論對象是誰都要傳福音
  2. 屬靈的能力是不能用錢買的
  3. 順服聖靈的結果是超乎想像的

應用

學習踏出自己的舒適圈去傳揚福音
